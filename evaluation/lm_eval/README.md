# Evaluation of MBPP and Humaneval
This folder contains the evaluation framework for MBPP and Humaneval.

## Quickstart
To install the toolkit, run:
```
pip install -e .
```

To run the evaluation, use:
```
bash eval_mbpp_humaneval.sh <model_path>
```
